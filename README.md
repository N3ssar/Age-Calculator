# Age Calculator

This is a simple web-based application that calculates the user's age based on the inputted day, month, and year. The application is responsive and user-friendly, providing feedback if invalid dates are entered.

## Features

- Input validation for day, month, and year
- Handles leap years
- Real-time age calculation upon form submission
- Error messages for invalid inputs
- Supports keyboard interaction for accessibility (Enter key to submit)

## Technologies Used

- **HTML**: For the basic structure of the web page.
- **CSS**: For styling the form and result display.
- **JavaScript**: For validating inputs and calculating the user's age.

## How to Use

1. Enter your birth date (day, month, year) into the input fields.
2. Click the "Calculate" button or press the "Enter" key.
3. Your age will be displayed below the form.

## Project Structure

Age Calculator/ │ ├── index.html # The main HTML file for the project ├── style.css # The CSS file for styling the page └── script.js # The JavaScript file for form validation and age calculation


## Usage Example

1. Input: Day: `12`, Month: `05`, Year: `1990`
2. Output: `You are 34 years old`

## Another Example

1. Input: Day: `29`, Month: `02`, Year: `2000` (Leap Year)
2. Output: `You are 24 years old`
